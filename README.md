# UV

I use uv, a modern Python package manager + virtual environment tool which simplifies making venv and adding python modules.


# Installation

To install all requirements from Python, run the following:

## UV

```bash
uv pip install -r requirements.txt
uv pip install -e ./modules/floris/
uv pip install -e ../modules/LandBOSSE/
```


## Pip

```bash
pip install -r requirements.txt
pip install -e ./modules/floris/
pip install -e ./modules/LandBOSSE/
```
